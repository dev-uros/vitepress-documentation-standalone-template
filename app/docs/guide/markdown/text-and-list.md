## Lists

List can be either ordered on unordered

### Ordered list

1. First item
2. Second item
3. Third item
4. Fourth item
5. Fifth item


### Unordered list

- First item
- Second item
- Third item
- Fourth item
- Fifth item


### Combined

- First item
  1. Section one
  2. Section two 
- Second item
  1. Section three
  2. Section four
- Fourth item
  1. Section five
     - Subsection five
